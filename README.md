# Breast Cancer Classification Streamlit App

This is a simple web app to predict breast cancer class (Malignant or Benign) using a pre-trained ML model with PCA.

## How to Run

1. Make sure you have Python installed (>=3.8)
2. Install dependencies:
   pip install -r requirements.txt
3. Put 'best_model_breast_cancer.pkl' in the same folder
4. Run the app:
   streamlit run app.py
5. Enter the 30 features and click 'Predict' to see the result.

## Files

- app.py : Streamlit web app
- best_model_breast_cancer.pkl : Pre-trained ML model
- requirements.txt : Dependencies
- README.md : Project info
